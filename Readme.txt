# EdTech: Öğrenci Başarı Tahminleyici

Bu proje, öğrencilerin çalışma süresi ve devamsızlık gibi verilerini kullanarak final notlarını (G3) tahmin eden bir yapay zeka uygulamasıdır.

## Özellikler
- Makine öğrenmesi modeli (Scikit-learn)
- Streamlit ile interaktif web arayüzü
- Gerçek zamanlı veri tahmini

## Nasıl Çalıştırılır?
1. Repoyu klonlayın.
2. Gerekli kütüphaneleri kurun: `pip install -r requirements.txt`
3. Uygulamayı başlatın: `streamlit run app.py`
